# Sesiunea 18 - Vizualizare de Date cu Matplotlib (Arbore Conceptual)

- 📌 Sesiunea 18: Matplotlib Basic
  ├── 🎯 Obiectiv Core: Introducere în vizualizarea de date elementară
  ├── ⚙️ Configurare Mediu
  │   ├── Instalare: `pip install matplotlib`
  │   └── Import standard: `import matplotlib.pyplot as plt`
  ├── 📈 Tipuri de Grafice Fundamentale
  │   ├── Grafic Liniar (`plt.plot`) -> Ideal pentru tendințe continue
  │   ├── Grafic cu Bare (`plt.bar`) -> Ideal pentru comparații între categorii discrete
  │   ├── Histogramă (`plt.hist`) -> Distribuția de frecvență a unei variabile continue
  │   └── Scatter Plot (`plt.scatter`) -> Corelații și dispersie între două variabile numerice
  ├── 🔲 Structuri și Layout-uri
  │   └── Subploturi (`plt.subplot`) -> Grile de grafice multiple în aceeași figură
  ├── 🎨 Personalizări Estetice de Bază
  │   ├── Etichete și Titluri (`plt.title`, `plt.xlabel`, `plt.ylabel`)
  │   ├── Stiluri de linii și marcatori (`color`, `linestyle`, `marker`)
  │   └── Legende și Caroiaj (`plt.legend`, `plt.grid`)
  └── 💾 Export Date
      └── Salvare fișiere (`plt.savefig`) -> Export în format PNG sau PDF
